Digital Mishnah Project

Description

This page contains links to sample output from ongoing work on the Digital Mishnah project. These files are static, but have been produced through early-stage implementations of Cocoon pipelines on Mishnah files (in TEI/xml) using XSLT and, for collation, a cocoon implementation of CollateX. (The CollateX Cocoon implementation is that of Gregor Middell and set up by Travis Brown of MITH.)
